# 🦷 Cabinet Dentaire - Application de Gestion

Application web complète pour la gestion de cabinet dentaire.

## ✨ Fonctionnalités

- 📊 **Dashboard** - Statistiques et vue d'ensemble
- 👥 **Gestion des patients** - Ajout, modification, suppression
- 📅 **Rendez-vous** - Planning et suivi
- 💰 **Paiements** - Gestion des règlements (espèces/chèques)
- 🩻 **Radiographies** - Upload et visualisation
- 📋 **Consultations** - Historique médical
- 💊 **Stock médicaments** - Gestion des produits
- 📱 **Responsive** - Fonctionne sur PC, tablette et mobile

## 🛠 Technologies utilisées

**Backend**
- Node.js + Express
- MySQL
- JWT pour l'authentification
- Bcrypt pour le hash des mots de passe

**Frontend**
- React 18
- Bootstrap 5
- Axios
- React Router DOM
